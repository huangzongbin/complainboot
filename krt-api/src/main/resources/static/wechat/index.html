投诉举报须知

一、投诉举报受理范围：

1. 赣州市保障性住房建设运营有限公司负责管理的保障房小区：渡口路小区、章江南苑公租房小区、阳光·吉泰新城小区、阳光·龙都嘉苑小区、阳光·金色春城小区、阳光·黄金家园小区、阳光·西域华城小区、阳光·和谐家园小区

2.违规承租方面：承租家庭将公共租赁住房转租的;承租家庭存在提供虚假证明材料等欺骗方式取得公共租赁住房的；租赁期内，承租人通过购买、受赠、继承等方式获得其他住房并不再符合公共租赁住房配租条件的；在公共租赁住房内存放危险物品或从事违法活动的。

3.扫黑除恶方面：承租家庭为黑恶势力，或房管工作人员、物业工作人员充当黑恶势力“保护伞”的情形。

4.保障房窗口服务方面：工作人员在受理办事群众咨询或办理相关业务时，不履行首问责任制、一次性告知制、限时办结制，态度冷漠、敷衍塞责、拖拉推诿，让办事群众来回跑、到处跑，不愿听取服务对象合理诉求；工作人员收受办事群众现金红包、微信红包、礼券、礼品等，接受办事群众安排的宴请和娱乐等其他吃拿卡要等行为。

5.小区物业管理存在的问题。


但以下情况除外：

1、不属于赣州市保障性住房建设运营有限公司管理职责范围的；

2、无明确的投诉举报对象或违规违法行为的；

3、投诉举报已在处理中，或者依法应当通过行政复议、诉讼、仲裁等法定途径解决的；


二、注意事项：

1、提倡实名投诉举报并填写相关联系方式。

2、请如实填写投诉举报表单，投诉举报内容力求详尽，对所投诉举报事件发生的时间、地点、主要证据、涉及人员等要交代明确。

3、标有“*”的为必填项目。

4、表单提交后三个工作日工作人员将联系回复受理情况。

5、投诉举报人应据实进行投诉举报，不得捏造事实、制造假证、诬告陷害他人。对借举报诬告陷害他人的，依照有关规定严肃处理；构成犯罪的，移送司法机关依法处理。

6、赣州市保障性住房建设运营有限公司保留删除恶意投诉举报单的权利。

7、赣州市保障性住房建设运营有限公司对投诉举报人提供的信息严格保密。
